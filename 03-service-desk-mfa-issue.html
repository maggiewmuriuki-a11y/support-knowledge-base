📅 Event Coordination Workflow

Document Type: Operations Workflow
Category: Event Coordination
Version: 1.0
Last Updated: February 2025
Author: Margaret Muriuki, Client Services Coordinator

📌 Overview

This document provides a structured workflow for planning and executing company events, including client meetings, webinars, and internal team engagements. It ensures all events run smoothly, on time, and within scope.

Applicable Events:

  • Client onboarding sessions
  • Quarterly business reviews (QBRs)
  • Internal team meetings and trainings
  • Webinars and virtual workshops

🗓️ Phase 1: Event Planning (4–6 Weeks Before)

1.1 Define the Event

  • Confirm event purpose, audience, and expected outcomes
  • Determine event format: in-person / virtual / hybrid
  • Set target date and time
  • Create SharePoint folder

1.2 Planning Tracker

Item Owner Deadline Status
Venue/Platform booking Coordinator Week 1 Done
Agenda draft Manager Week 2 In Progress
Invitations sent Coordinator Week 2 Pending
Materials prepared Presenter Week 3 Pending
Tech check IT Support Week 4 Pending

📧 Phase 2: Communication & Invitations (3 Weeks Before)

Send Invitations

  • Draft invitation email
  • Send via Outlook with calendar invite
  • Log invitees in HubSpot

Track RSVPs

  • Update RSVP tracker
  • Send reminders to non-responders
  • Confirm final attendee count

🔧 Phase 3: Pre-Event Preparation (1 Week Before)

Logistics Confirmation

  • Confirm venue or platform
  • Ensure materials are ready
  • Distribute agenda

Technical Setup

  • Submit ServiceNow request
  • Test Teams features
  • Confirm presenters are ready

🎯 Phase 4: Event Day Execution

Before the Session

  • Log in 30 minutes early
  • Test audio/video
  • Prepare attendance sheet

During the Session

  • Welcome attendees
  • Introduce agenda
  • Monitor chat
  • Take meeting minutes

Technical Issues Protocol

Issue Action
Presenter cannot connect Use backup presenter
Audio/video failure Switch to phone audio
Recording stops Restart recording
Platform drops Send backup link

📝 Phase 5: Post-Event Follow-Up (Within 48 Hours)

  • Send follow-up email
  • Share minutes, slides, and action items
  • Log tasks in tracker
  • Upload recording to SharePoint
  • Send feedback form

🛠️ Tools Used

  • Microsoft Outlook
  • Microsoft Teams
  • SharePoint
  • HubSpot
  • ServiceNow
  • Excel
